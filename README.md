### Post

1. Create post inside `_posts` with format file with `YYYY-MM-DD-TITLE.md`
2. Add formatter 

```

   ---
    layout: post
    title:  "Parsing JSON with Ruby"
    date:   2013-12-23 00:18:23 +0700
    categories: [ruby]
    ---
```
3. Add category if not present else skip.
