### Vim

Vim - is Vi Improved editor for programmers.

### Some commands
```
# while editing file
:help
# to route between helpful commands that vim provides
# help ranges from tutorial to more deep vim specific
# manuals.

# shows manual page for vim and their arguments for 
# opening file
man vim

```
